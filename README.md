# AMP NestJs Demo app


This  this app using Next.js and the AMP feature. 


## How to use

Clone 

```bash
git clone git@github.com:GeekRishabh/demo-amp.git
```


### Dev Mode

```bash
npm run dev
```

### Prod Build Mode

```bash
npm run build
```

### Start Prod Mode

```bash
npm run start
```


PS: Component file & CSS file are copied from internet
